Linux相关

25 Mar 2011
  • Debian
  • Linux
  1. 查询进程：ps afx | grep nginx
  2. 列出文件夹内容：ls，ls -l，ll
  3. 软件安装相关：apt-cache search mysql，apt-get update，apt-get install mysql
  4. 查找文件：find / -name filename.txt
  5. root授权：sudo su -
  6. 文件内容查找：cat syslog |grep  searchcontent > resultfile.log
  7. 给予文件可执行权限：chmod +x filename
  8. memcached：将启动时候的-l 参数设成0.0.0.0就不会遇到某些IP无法访问memcache的问题
  9. ll命令：打开 ~/.bashrc，找到 #alias ll=’ls -l’，去掉前面的#就可以了
  10. 查询软件版本：dpkg –l |grep 软件名
  11. scp：scp  -P 22 /root/abc.zip  root@192.168.1.102:/abc/
  12. 查询一个文件中，包含文本为xxx，且第七列内容为最小5个的那几行：grep ' xxx' syslog | awk '{print $7}' |sort -n |tail -5
  13. 查询一个文件中，不包含文本为xxx，且第七列内容为最小5个的那几行：grep -v ' xxx' syslog | awk '{print $7}' |sort -n |tail -5
  14. 查询一个文件中，包含文本为xxx，且第七列内容为最大5个的那几行：grep ' xxx' syslog | awk '{print $7}' |sort -n |head -5
  15. 查询一个文件中，不包含文本为xxx，且第七列内容为最大5个的那几行：grep -v ' xxx' syslog | awk '{print $7}' |sort -n |head -5
  16. 查询第七列的总和与平均值：awk '{print $7}' | perl -e 'while(<STDIN>){chomp;$i++; $a+=$_} print "$i\n", $a/$i,"\n"'
  17. 查询某种进程的总数：ps axu|grep 进程名 |grep -v grep |wc -l
  18. 清空syslog文件：> /var/log/syslog
  19. 显示所有全局变量export，全局变量文件/etc/profile
  20. 改变开机启动项命令：chkconfig --list | --del name
