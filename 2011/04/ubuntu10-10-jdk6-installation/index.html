在Ubuntu10.10上安装JDK6

26 Apr 2011
  • Java
  • Linux
  • Ubuntu

Ubuntu10.10官方的maverick包里是不包含sun的java软件的，如果想安装sun-java6系列软件的话，我们需要自己添加一个源：

添加文件：/etc/apt/sourses.list.d/canonical.list
内容为：

deb  http://archive.canonical.com/ubuntu maverick partner

接下来运行：

apt-get update
apt-get install sun-java6-jdk

OK，完结了，其他的apt都会帮你搞定。以jdk为目标软件安装的话，jre和java-bin都会一并安装好的，而且不用自己配置任何东西。

附上canonical软件池地址：http://archive.canonical.com/pool/partner
