使用samba搭建简单的文件共享服务器

03 Nov 2011
  • Debian
  • Linux
  • Samba

废话不多说，直接上步骤。首先当然是安装：

[codesyntax lang="bash"]

apt-get install samba

[/codesyntax]

然后就是配置了，配置文件在：/etc/samba/smb.conf。内容分两块，一块是samba的全局配置，另一块是你的共享目录的配置。

全局配置：

workgroup：这个配置不重要，只要你的机器能访问到samba服务器，那么正常的使用是不会受到影响的。
security：设成share，表示用户不需要帐号及密码可以登陆Samba服务器。

目录配置：

在“==== Share Definitions ====”注释下面的方括号里，填上你的共享目录显示名，例如[Shared Folder]。然后在下面填写下述的配置信息：

browseable = yes
path = /home/shares（填上你需要共享的目录）
public = yes
writeable = yes
read only = no
guest ok = yes
create mask = 0777
force create mask = 0777
directory mask = 0777

好了，这样你刚才配的那个路径就是完全开放的了，当然要注意这个共享目录是完全没有权限限制的，需要安全的东西当然不能放这里。
