eclipse使用技巧

23 Dec 2012
  • eclipse
  • IDE

这个礼拜花了不少时间看了下Sublime Text 2这款IDE，感觉还是不如意，虽然它有它的优点，但是和eclipse比起来，致命的缺点却还是让我留在了eclipse的怀抱。这时突然转念一想，既然无法离开eclipse，那么就学着更好地使用eclipse吧。于是有了这篇文章。

说到eclipse的技巧，必须要掌握的，肯定就是eclipse的快捷键了，使用快捷键，才能帮助我们更好地工作：

  • ⌘D：删除当前行
  • ⌘⇧R：快速打开项目中的某个文件（每天要用无数次的快捷键）
  • ⌘⇧T：快速打开某个类的定义（与上面不通的是，不是文件，而是类）
  • ⌘O：快速outline（方便在全屏编辑的状态下快速定位）
  • ⌘⌥R：快速重命名
  • ⌘L：快速到某一行
  • ⌘← 或 ⌘→：到行首或行末
  • ⌥← 或 ⌥→：到下个或前个单词
  • ⌘E：快速在已经打开的编辑页面中切换
  • ⌘[ 或 ⌘]：前进、后退到上个、前个编辑点
  • ⇧↩ 或 ⌘⇧↩：在当前行下、上，创建一个空白行
  • ⌥↑ 或 ⌥↓：将当前行的内容上、下移动一行
  • ⌘⌥↑ 或 ⌘⌥↓：将当前行的内容上、下复制一行
  • ⌘T：查看当前类的继承关系
  • ⌃Q：定位到最后的编辑位置
  • ⌘/：批量注释、反注释
  • ⌥/：自动补完
  • ⌘J 或 ⌘⇧J：打开快速正向、反向增量查找，按下⌘J之后，你输入的所有内容都会作为查找条件进行查找，且连续按⌘J会按顺序一个个查找下去
  • ⌘F：打开查找与替换面板
  • ⌥F：打开文件查找面板（这个是我自己设定的）
  • ⌘K 或 ⌘⇧K：查找下、上一个匹配
  • F4：打开类型层次结构
  • ⌥↩：打开属性面板
  • ⌘⇧P：定位到对应的匹配符(譬如{})
  • ⌃T 或 ⌃⇧T：切换到下、上个tab（这个是我自己设定的）

基本上有了这些快捷键，你就不怎么需要一直靠鼠标来进行操作了，大大提高你的工作效率
