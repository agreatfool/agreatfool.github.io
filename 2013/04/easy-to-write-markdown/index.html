舒适地写文章 - Markdown

17 Apr 2013
  • Markdown

实在是找不到适合放这篇文章的分类目录，也不希望因为单独第一篇文章就创建一个分类目录。因为使用的方式和IDE比较接近，反正都是输入内容，然后输出想要的格式，所以就放IDE里了，囧。

第一次接触到Markdown是在使用Trello的时候，trello的description还有comment两个输入栏默认就是markdown输入格式。那时候第一次编辑markdown，也不知道语法，写出来的东西感觉好怪异，总是有各种特殊格式。后来在看很多开源项目的readme的时候，发现都是README.md，都是markdown格式。就兴起了尝试下的想法。

尝试了才有资格说，markdown不愧于它的宣传：“易于阅读，易于书写”。写起来一点难度都没有，而且非常符合人类自然的书写方式，没有像HTML那样的过多格式，看起来也很舒服，已经能满足绝大多数的富语言表达。

学习的过程也不细说了。给点有用的资料。

Markdown官网：http://daringfireball.net/projects/markdown/

Markdown的语法中文翻译：http://markdown.tw/

Mac上超酷的Markdown编辑器Mou：http://mouapp.com/
