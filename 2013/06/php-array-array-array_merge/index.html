PHP array + array & array_merge

06 Jun 2013
  • PHP

PHP的数组融合一般有两种做法，一种是直接使用加号相加，另一种则是使用array_merge函数进行相加，两者之间有点区别：

  • 相加会证数组中的自然index不被重置
  • 相加方法中，被加的数组内的值不会被覆盖
  • merge函数中的自然index会被重置
  • merge函数，无所谓被merge和merge关系，越后面的array参数，其值，会覆盖较前面的array参数相同键的值

例子：

数组1：

[codesyntax lang="php"]

$arr1 = array(
    10 => 'valueof10',
    11 => 'valueof11',
    'key1' => 'dataofkey1',
    'key2' => 'dataofkey2',
    'array' => array(
        'key1' => 1,
        'key2' => 'abc',
        'key3' => array(
            1, 2, 'name' => 'peter'
        ),
    ),
);

[/codesyntax]

数组2：

[codesyntax lang="php"]

$arr2 = array(
    10 => 'newvalueof10',
    11 => 'newvalueof11',
    'key1' => 'newdataofkey1',
    'key3' => 'newdataofkey3',
    'array' => array(
        'key1' => 2,
        'key2' => 'defg',
        'key3' => array(
            1, 2, 'name' => 'jonathan', 'gender' => 'male'
        ),
    ),
);

[/codesyntax]
