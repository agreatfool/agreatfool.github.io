V8 release v6.7

2. 摘要翻译

每柳州，我们会创建一个新的V8分支作为我们的release流程的一部分。每个分支是在Chrome的Beta milestone之前，从V8的 Git master分支上分出来的。今天我们很高兴地宣布我们最新的分支，V8版本6.7，在几周之后同步和Chrome67稳定版本一起release之前将会保持beta装填。V8 6.7充满着各种对开发者有帮助的好东西。这篇文章将会提供一个release的预览。

JavaScript language features

V8 6.7实装了BigInt支持，并默认开启。BigInts是JavaScript中的一个新数字原始类型，能表示任何精度的数字。请阅读the Web Fundamentals article on BigInt这篇文章来获得更多关于BigInt的信息，关于BigInts是如何在JavaScript中被使用的，并了解V8中实现的更多细节信息。

Untrusted code mitigations

在V8 6.7中，我们带来了更多针对”side-channel漏洞的解决方案“来防止信息被泄露给未受信任的JavaScript以及WebAssembly代码。

V8 API

请使用git log branch-heads/6.6..branch-heads/6.7 include/v8.h来获得API改动的列表。
